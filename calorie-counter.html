Calorie Calculator

What is a Calorie Calculator?

Here, we are using the Mifflin St Jeor calorie calculator, which calculates the BMR. The BMR (Basal Metabolic Rate) is an estimate of how many calories you'd burn if you were to do nothing but rest for 24 hours. It represents the minimum amount of energy needed to keep your body functioning, including breathing and keeping your heart beating.

Your BMR does not include the calories you burn from normal daily activities or exercise.

How do we calculate it?

The formula is based from the famous Mifflin St Jeor equation.

Men: BMR = 5 + ( 10.0 x weight in kilos ) + ( 6.25 x height in cm ) - ( 5.0 x age in years )

Women: BMR = -161 + ( 10.0 x weight in kilos ) + ( 6.25 x height in cm ) - ( 5.0 x age in years )
